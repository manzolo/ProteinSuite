# Introduction

## Teaching level

Protein Suite introduces proteins, sequences, and structures with tools that run in the browser.

## Scientific level

Use the Catalog from real identifiers, the Lab for estimated properties, and FASTA Manager to clean sequences.

## Tool links

- Protein Catalog: `#/catalog`
- Protein Explorer: `#/explorer`
- Protein Lab: `#/lab`
- FASTA Manager: `#/fasta`
