# PDB

## Livello didattico

PDB identifica strutture macromolecolari depositate nel Protein Data Bank.

## Livello scientifico

Nel Catalogo gli ID PDB locali sono cross-reference UniProt validate con RCSB Data API.

## Collegamenti agli strumenti

- Catalogo proteine: `#/catalog`
- Protein Explorer: `#/explorer`
- Protein Lab: `#/lab`
- FASTA Manager: `#/fasta`
