# Idrofobicità

## Livello didattico

L’idrofobicità descrive la tendenza di residui o regioni a evitare l’acqua.

## Livello scientifico

Il Lab calcola GRAVY e profilo a finestra con scala Kyte-Doolittle, utile per regioni transmembrana o core idrofobici.
